Sobre este fluxo de automação:
Através de fluxo você pode realizar o download da 2 via da fatura no site da COPEL usando o CPF

Para realizar esse download você deve ter em mãos os dados de:

Numero COPEL
CPF da Conta
Senha para acesso ao portal
Autor: AutomationEdge
Tipo: Web
